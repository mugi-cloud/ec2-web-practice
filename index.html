AWS EC2 Web Server Project

このWebサイトは、AWS EC2上に構築したWebサーバーの動作確認用ページです。

Apacheを使用し、インターネットからアクセスできる環境を構築しました。

構成情報

  • クラウド：AWS
  • サービス：EC2
  • OS：Amazon Linux 2023
  • Webサーバー：Apache
  • リージョン：ap-northeast-1（Tokyo）

ポイント

  • EC2上にWebサーバーを構築
  • Apacheを用いてWebページを公開
  • Elastic IPを設定し、固定IPでアクセス可能にした
  • セキュリティグループでHTTP通信を許可

この成果物について

AWSの学習内容をアウトプットするために作成しました。

実際に環境構築を行う中で、インフラの仕組みへの理解が深まりました。 今後はより実践的な構成にも挑戦していきたいと考えています。
